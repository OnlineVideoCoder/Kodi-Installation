Kodi Full Setup - Amazon Firestick - Without Real-Debrid Educational Purpose Only

Installing Downloader and App Starter

  1. Install Downloader from Amazon App Store
  2. Enter Downloader / Allow Access / OK
  3. Enter URL - Bit.ly/app-starter / Click Install on Modifiy System / Click Done / Click Install APK / Click Install / Return to Home

Downloading The Crew

  • Enter Downloader/ Enter URL - https://https://onlinevideocoder.github.io/Kodi-Installation/ *This Page* / Click Go / Click on "The Crew Download Link" / Click on repository.thecrew-*.*.*.zip *To Install*

Downloading Elementum

  • Enter Downloader/ Enter URL - https://https://onlinevideocoder.github.io/Kodi-Installation/ *This Page* / Click Go / Click on "Elementum Website" / Go to Elementum Downloads Header and Find Universal All-in-one(plugin.video.elementum-*.zip) *Click To Install*

Downloading Skin

  • Enter Downloader/ Enter URL - https://https://onlinevideocoder.github.io/Kodi-Installation/ *This Page* / Click Go / Click on [Preferred] Skin

Downloading Kodi

  1. Go to Settings / My Fire TV / About / Fire *Device Name* Select Device 5 Times to enter Developer Mode
  2. Go to Settings / My Fire TV / Developer Options / Install Unknown Apps / Appstarter and Downloader / On
  3. Open Appstarter / Updates *On Left Hand Side*/ Kodi / Install
  4. Click Install / Done / Return to Home

Installing Skin

  1. Open Kodi / Click on the Cog at the top / File Manager / Add Source / Browse / Externmal Storage / Download / Downloader / OK / OK / Backout to Main Kodi Screen
  3. Open Kodi / Click Continue / Click Allow / Settings / File Manager / Add Source / Browse / Externmal Storage / Download / Downloader / OK / OK / Backout to Main Kodi Screen
  4. Go to Settings / Add-ons / Install From Zip File / Click Settings oon pop up / Toggle On Unknown Sources on Kodi / Yes / Click Back Button / Go back to install from zip file / Downloader / repository.thecrew-*.*.*.zip / Wait for Add-on Enabled Notification / Install From Repository / The Crew Repository / Video Add-ons / The Crew Wizard / Install / Wait for Add-on Enabled Notification / Dismiss / Continue
  5. Build Menu / Scroll Down to Franks Build / Click on Fresh Install / Yes / *This should close down Kodi*
  6. *Optional This is to Pin Kodi to Front Of Home Page - On Firestick go to Apps / Click the 3 Lines Button on the remote / Move to front*

Installing Kodi and The Crew

  1. Go to settings / Applications / Manage Installed Applications / Kodi / Clear Data and Clear Cache *Check If Data and Cache = 0 Byte*/Return to Home
  2. Open Appstarter / Updates / Check For Updates On Kodi *Should be all up to date*/ Return to Home
  3. Open Kodi / Click Continue / Click Allow / Settings / File Manager / Add Source / Browse / Externmal Storage / Download / Downloader / OK / OK / Backout to Main Kodi Screen
  4. Go to Settings / Add-ons / Install From Zip File / Click Settings oon pop up / Toggle On Unknown Sources on Kodi / Yes / Click Back Button / Go back to install from zip file / Downloader / repository.thecrew-*.*.*.zip / Wait for Add-on Enabled Notification / Install From Repository / The Crew Repository / Video Add-ons / The Crew Wizard / Install / Wait for Add-on Enabled Notification / Dismiss / Continue
  5. Build Menu / Scroll Down to Franks Build / Click on Fresh Install / Yes / *This should close down Kodi*
  6. *Optional This is to Pin Kodi to Front Of Home Page - On Firestick go to Apps / Click the 3 Lines Button on the remote / Move to front*

Installing Elementum

  • Open Kodi / Click System / Add-ons / Install From Zip File / Downloader / plugin.video.elementum-*.zip / Wait for Add-on Enabled Notification / Install From Repository / Elementum Repository / Video Add-ons / Elementum / Install / Wait for Add-on Enabled Notification / Dismiss / Continue

Installing Skin

  1. On Main Screen Hover System / File Manager / Left Side - Downloader / *Skin.zip* / *Skin Folder* / Addon / Right Side - home / addons / Left Side - Hold Select on skin.* / Copy / Click Back / Addon-data / Right Side - Click Back / Userdata / addon_data / Left Siide - Hold Select on skin.* / Copy / Keep Clicking Back to Main Screen
  2. System / Add-ons / My Add-ons / Look and Feel / Skin / Click on Skin / Enable / Use / Keep Clicking Back to Main Screen

You Have Successfully Installed Kodi With The Crew And Elementum Repositorys with a Custom Skin
