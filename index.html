Kodi Full Setup - Amazon Firestick - Without Real-Debrid Educational Purpose Only

Installing Downloader and App Starter

  1. Install Downloader from Amazon App Store
  2. Enter Downloader / Allow Access / OK
  3. Enter URL - Bit.ly/app-starter / Click Install on Modify System / Click Done / Click Install APK / Click Install / Return to Home

Downloading The Crew

  • Enter Downloader/ Enter URL - https://https://onlinevideocoder.github.io/Kodi-Installation/ *This Page* / Click Go / Click on "The Crew Download Link" / Click on repository.thecrew-*.*.*.zip *To Install*

Downloading Elementum

  • Enter Downloader/ Enter URL - https://https://onlinevideocoder.github.io/Kodi-Installation/ *This Page* / Click Go / Click on "Elementum Website" / Go to Elementum Downloads Header and Find Universal All-in-one(plugin.video.elementum-*.zip) *Click To Install*

Downloading Skin

  • Enter Downloader / Enter URL - https://https://onlinevideocoder.github.io/Kodi-Installation/ *This Page* / Click Go / Click on [Preferred] Skin

Downloading Kodi

  1. Go to Settings / My Fire TV / About / Fire *Device Name* Select Device 5 Times to enter Developer Mode
  2. Go to Settings / My Fire TV / Developer Options / Install Unknown Apps / Appstarter and Downloader / On
  3. Open Appstarter / Updates *On Left Hand Side*/ Kodi / Install
  4. Click Install / Done / Return to Home

Installing Kodi and The Crew

  1. Go to settings / Applications / Manage Installed Applications / Kodi / Clear Data and Clear Cache *Check If Data and Cache = 0 Byte*/Return to Home
  2. Open Appstarter / Updates / Check For Updates On Kodi *Should be all up to date*/ Return to Home
  3. Open Kodi / Click Continue / Click Allow / Settings (Cog) / Add-ons / Install From Zip File / Click Settings on pop up / Toggle On Unknown Sources on Kodi / Yes / Click Back Button / Go back to install from zip file / External Storage / Download / Downloader / Click on repository.thecrew-*.*.*.zip / Wait for Add-on Enabled Notification / Install From Repository / The Crew Repository / Program Add-ons / The Crew Wizard / Install / Wait for Add-on Enabled Notification / Dismiss / Continue
  4. Build Menu / Scroll Down to Franks Build / Click on Fresh Install / Yes / *This should close down Kodi*
  5. *Optional This is to Pin Kodi to Front Of Home Page - On Firestick go to Apps / Click the 3 Lines Button on the remote / Move to front*

Installing Elementum

  • Open Kodi / Click System / Add-ons / Install From Zip File / External Storage / Download / Downloader / plugin.video.elementum-*.zip / Wait for Add-on Enabled Notification / Install From Repository / ElementumOrg Repository / Click OK / ElementumOrg Repository / Video Add-ons / Elementum / Install / Wait for Add-on Enabled Notification / Yes / Go back until you see home screen

Installing Skin

  1. On Main Page Click System/ Add-ons / Install from Zip File / External Storage / Download / Downloader / AetherView-Skin-V* / Wait for Notification / Click Yes / Click Yes

Installing Multiple Skins

  1. On Main Page Click System/ Add-ons / Install from Zip File / External Storage / Download / Downloader / Aetherview-Skin-V* / Wait for Notification / Click No / My Add-Ons / Look and Feel / Skin / AetherView *

This allows you to swap between different skins prioritising different Repository's (This means the shortcuts on movies would go to the specific repository e.g the Crew or Elementum)
